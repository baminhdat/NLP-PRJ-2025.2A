# NLP-PRJ-2025.2A
#File zip này bao gồm 5 file source codes khác nhau cho các approach của nhóm:
- visobert-RAG là source code cho approach Contrastive Learning for Retrival.
- lora-visobert-vigoemotions là source code cho approach LoRA Finetuning
- emotions-embedding-learning là source code cho approach Learning Emotions embeddings
- vinallama-inference là source code để thực hiện inference trên model VinaLlama
- final-results-parsing-knn-and-evaluation là source code để thực hiện phương pháp đơn giản knn từ kết quả của visobert-RAG, emotions-embedding-learning và tìm thresholds, đánh giá kết quả prediction của các phương pháp
